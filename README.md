# B HTML EXAMPLE 1

Demonstration of how to write html files and preview them from vscode.

## Questions

- do all html tags need to be written the same?
- is capitalization important?
- can you use spaces in names?
- does index.html have to be named like that?
- how to make new files in vscode?
- how do I [deploy to github pages](https://docs.github.com/en/pages/getting-started-with-github-pages/creating-a-github-pages-site)

- X explore how to write html syntax (what's important?)

- X create 2 levels of headings
